# Dominick Heugas Portfolio

A clean personal portfolio website for Dominick Heugas, an aspiring web developer focused on lead-generation systems for local service businesses.

## Overview

This static website showcases three fictional portfolio demos built around local service business marketing workflows:

- Responsive landing pages
- Quote and booking forms
- UTM campaign tracking
- localStorage lead capture
- Business owner dashboard UI concepts

## Featured Projects

1. **Bronx Pro Renovations**  
   Contractor lead-generation landing page with quote form, UTM tracking, localStorage lead capture, and owner dashboard.

2. **Bronx Elite Auto Detailing**  
   Mobile auto detailing booking page with package interest tracking, UTM source capture, and business owner lead dashboard.

3. **Bronx Fresh Cleaning Co.**  
   Cleaning company quote-request landing page with service tracking, property type tracking, UTM attribution, and dashboard reporting.

## Files

- `index.html` - Page structure and portfolio content
- `styles.css` - Responsive visual design
- `script.js` - Mobile navigation and dynamic footer year
